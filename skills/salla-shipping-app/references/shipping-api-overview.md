# Salla Shipping API — Overview

Salla Shipping API lets you manage and track shipments and orders for Salla stores. Integrate with a store's shipping process to handle multi-order, multi-shipment, and multi-branch scenarios.

## Capabilities

- Retrieve shipment information
- Create shipment labels
- Set tracking IDs
- Manage multiple orders, shipments, and branches

## Prerequisites

- Verified account on [Salla Partners Portal](https://salla.partners)
- A Shipping App created on the Partners Portal (must be **Public** — Shipping Apps cannot be Private)
